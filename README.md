# Anidb
An Unofficial AniDB API

## Install
```composer require gloire/anidb```
### Dependencies
- [Cloudinary](https://github.com/cloudinary/cloudinary_php)
- PHP 7.0.0
## Features
- Anime
  - Title
  - Image
  - Tags
  - Start & End Date
  - Rating
  - Description
  - Favourites
  - Rank
  - Completed
  - Watching
  - Plan
  - Drop
  - Directly Related Animes
  - Similar Animes
  - Cast
  - Staff
  - Characters
